Definition

Traditionally, a combo-box is a combination (get it?) of a drop-down list or list box and a single-line editable textbox, allowing the user to either type a value directly into the control or choose from a list of existing options.

Problem being solved

If the use case allows for it, allowing a user to provide their own value can increase data integrity assuming that the user knows what they're doing. For expert users, this can add real value while your novice users will appreciate having a list of default options.

Why you would use it

  • If your user is able to select from a list of predefined values, OR provide a new value that is not already in the list.

Why you would not use it

  • This might seem obvious, but do not confuse this component with a standard drop-down list. If the use case you are trying to solve for does not allow for unique entries then a combo-box should not be used.

Best Practices

  • In our experience, a combo-box is not a commonly used component so make sure that your use case actually allows for a user to enter their own value.
